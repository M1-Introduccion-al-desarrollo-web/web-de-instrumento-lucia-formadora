Project: Piono

This project is an interactive web page that features a virtual piano, developed by coders from Factoría F5 as part of one of the assignments during our course.

The page allows users to play notes and chords directly from the browser, with keys that respond to mouse and keyboard input. It's ideal for beginners and musicians to practice melodies in a simple, accessible way.
