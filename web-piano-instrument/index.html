Project: Piono

This project is an interactive web page that features a virtual piano, developed by coders from Factoría F5 as part of one of the assignments during our course. The page allows users to play notes and chords directly from the browser, without the need for a physical piano. It is designed to be easy to use, with visual keys that respond to both mouse clicks and computer keyboard input. It is an ideal tool for beginners who want to learn, as well as for musicians who want to experiment or practice melodies in a simple and accessible way.
